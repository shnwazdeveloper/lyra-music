# Code of Conduct

## Our Pledge

Lyra Music aims to be a welcoming project for everyone. Contributors and maintainers are expected to communicate with respect and keep the project space helpful, safe, and professional.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different viewpoints and experiences.
- Accept constructive feedback.
- Focus on what is best for the project and its users.
- Show patience and empathy toward other contributors.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Sexualized language or unwanted attention.
- Trolling, insults, or derogatory comments.
- Publishing private information without permission.
- Any conduct that would be inappropriate in a professional community.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that do not follow this Code of Conduct.

To report a concern, contact the project maintainer through GitHub or Telegram:

# Contributing to Lyra Music

Thanks for helping improve Lyra Music.

## Report Bugs

Open an issue at:

https://github.com/shnwazdeveloper/lyra-music/issues

Please include:

- App version
- Android version
- Device model
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots or logs if useful

## Suggest Features

Open an issue and describe:

- What problem the feature solves
- How the feature should work
- Any examples or screenshots that explain the idea

## Pull Requests

1. Fork the repository.
2. Create a feature branch.
3. Keep changes focused.
4. Build the app before submitting.
5. Update docs when behavior changes.

Build command:

```powershell
.\gradlew.bat :app:assembleUniversalDebug
```

## Code Style

- Follow the existing Kotlin and Jetpack Compose style.
- Keep UI changes consistent with Material You patterns.
- Avoid unrelated formatting-only changes.
